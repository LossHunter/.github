# LossHunter 🎯

LossHunter팀 저장소에 오신 것을 환영합니다!
KT-Cloud TeckUP 생성형 AI 1회차 2팀 LossHunter입니다.
기본 프로젝트부터 시작해서 심화 프로젝트까지 개발한 코드들을 정리한 저장소입니다.

## 📚 저장소 소개

### [서비스](https://github.com/LossHunter/Vive-Trading)
Vive-Trading 서비스의 BE, FE의 코드 저장소입니다. FastAPI + React 조합으로 개발하고 있습니다.

### [AI](https://github.com/LossHunter/AI)
인공지능 및 머신러닝 관련 프로젝트를 담고 있는 저장소입니다. Google Colab에서 vLLM을 활용해 각 model을 서빙합니다.

### [Backtesting]
추후 공개 예정
개발한 LLM의 테스트 시뮬레이션 프로그램 입니다. 과거 기간동안 자료를 입력하여 테스트를 진행합니다.
LLM의 성능평가 또한 포함하여 개발하고 있습니다.
